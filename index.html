CalcDown — Demos

Run: make demo

Demo 1 — Markdown scratchpad

Editable source + parse/eval + simple chart rendering. Includes chart type switch (spec/line/bar).

Default: docs/examples/mortgage.calc.md

Demo 2 — Inputs → Cards

Simple inputs form that drives a standardized cards view.

Default: docs/examples/savings.calc.md

Demo 3 — Tables + Layout

Tabular data input, computed outputs, and standardized layout/table/cards views.

Default: docs/examples/invoice.calc.md

Demo 4 — Full View Renderer

Browse examples and render standardized cards, table, chart, and layout views.

Uses: multiple examples under docs/examples/

Demo 5 — External Data Sources

Loads data.source CSV/JSON tables in the browser, verifies sha256, then evaluates and renders views.

Default: docs/examples/invoice-external.calc.md

Specs: CalcDown 0.7 · stdlib 0.7
